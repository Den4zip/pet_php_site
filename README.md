# pet_php_site
Это пет проект современного деплоя сайта, написанного на php с использованием базы данных Postgresql и веб-сервера nginx.
Технологии:
    1.[![PHP](https://img.shields.io/badge/PHP-8.2-777BB4.svg?logo=php&logoColor=white)](https://www.php.net/)
    2.[![Nginx](https://img.shields.io/badge/Nginx-Stable-009639.svg?logo=nginx&logoColor=white)](https://nginx.org/)
    3.[![Docker Compose](https://img.shields.io/badge/docker%20compose-animated-blue.svg)](https://docs.docker.com/compose/)
    4.[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-15-336791.svg?logo=postgresql&logoColor=white)](https://www.postgresql.org/)
Этапы проектирования:
    1.Инициализация git-репозитория.
    2.Разработка сайта на PHP.
    3.Написание конфига для nginx и .* файлов.
    4.Написание Dockerfile и docker-compose.yml.
    5.Развёртывание проекта.
